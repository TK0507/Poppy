section-1 更新されたシグネチャ {

    fitメソッドのシグネチャが更新されました。

    新たにpoppy_sampleモジュールが追加され、
    要素であるSampleクラスを受け取るようになりました。

    Sampleクラスはtextプロパティとscoreプロパティを持ちます。

    fit2メソッドがModelに追加されました。

    これらの変更についての注意事項としては、
    fitメソッドのシグネチャ更新は破壊的変更であるということです。
    ただし、引き続きfitメソッドは単一のデータセット読み込みに利用されます。
}

section-2 今回新たに追加された命名について {

    今回新たに追加されたエンティティは以下の通りです。

    > Sample
    > Model.fit2
    > Model.save
    > Model.load

    これらの命名は、前回同様シンプルで分かりやすい命名であり、ほかの名前と衝突しません。
    そのため、特に理由のない限りこれらの命名の変更は非推奨です。
}
